# ComicBookTracker

This app was created to help comic readers keep better track of their comic collection, comics they have read, and comics they want to read. 

To run this app, ensure SQLite is installed.

If it is not, run this command: sqlite3 Comics.db

From the SQL prompt there, you can create the Comics table and populate the database using the queries in the sql-queries folder.

Once the database is functional, you can run the application using this command: dotnet run

Happy Comic Reading :)
